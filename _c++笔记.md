14. 如何产生随机数？熵池”

    1. 计算机里面有一个池子，池子里面有数字，各个数字之间的关系很难找到。但是，一旦找到了这个池子，里面的数字每次运行都是一样的，顺序也是一样的。每调用一次都相当于从池子中取出一个数字
    2. 头文件是<stdlib.h>
    3. 为了真正随机，需要每次运行的池子是不一样的，所以用时间！！！头文件<time.h>    
    4. srand(time(NULL))
    5. int x = rand() //每次运行都不一样
